How to use

  1. Choose CSV → JSON or JSON → CSV.
  2. Upload a file or paste your data directly into the box.
  3. Click Convert, then copy or download the result.

🔒 Your data is processed entirely in your browser. It is never uploaded to any server. For JSON → CSV, your JSON should be an array of objects (or a single object).
